# Installation

## Prerequisites

PyGCL needs the following packages to be installed beforehand:

* Python 3.8+
* PyTorch 1.9+
* PyTorch-Geometric 1.7
* DGL 0.7+
* Scikit-learn 0.24+
* Numpy
* tqdm
* NetworkX

## Manual Installation

To install PyGCL with [`pip`](https://pip.pypa.io/en/stable/), simply run:

```{code-block}
pip install .
```

Then, you can import `GCL` from your current environment.
